## Installation

```sh
git clone https://github.com/BenReichwein/TaskMe.git
cd client
npm install
```

## Quick Start - SERVER
```
# Start Development
go run .

### Make a .ENV in /server (check out .env.template)

DB_URI="MONGODB URI (insert)"
DB_NAME="TaskMe"
DB_COLLECTION_NAME="todolist"
```

## Quick Start - CLIENT
```
# Start Development
npm run dev

# Build API
npm run build

# Start API
npm run start
```
